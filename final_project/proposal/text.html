Final Project Proposal: Task board

Topic:

My productivity website will allow users to create and use task board to organize their daily activities

Audience:

The intended audience are people interested in staying organized with their work

Design and style:

  • goal: the design goal is to be sleek and modern. The app should look like it belongs in a catalog of other current-day apps
  • color scheme: I would like the app to be primarily in night mode with primarily colors dark grey and white
  • inspirations:
    • https://asana.com/
    • https://slack.com/
    • https://discord.com/

scope: 4 pages

  • sign up page: users create an account with a username and password
  • log in page: users login to their task board
  • task board page: page with most of functionality of site; users keep track of tasks here
  • landing page: first page users see when opening the page

database design

there will be 3 tables

  • users
  • tasks
  • categories

each table will have the fields:

  • users: id, username, password
  • tasks: id, name, datecreated, completed, category
  • categories: id, namea

source:

all data will be generated by users when they sign up and create tasks in the task board
